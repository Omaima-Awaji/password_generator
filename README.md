# Password Generator 🔐

A simple command-line password generator built with Python.

## Features
- Choose custom password length
- Select character types: lowercase, uppercase, numbers, symbols
- Generate multiple passwords in one session

## How to Use
Run the script:

python password_generator.py

Then follow the prompts:
1. Enter the desired password length
2. Type which character types to include (e.g. lowercase uppercase numbers symbols)
3. Enter YES to generate another or NO to exit

## Example

How long the password should be? 12
What character types should be included? lowercase uppercase numbers
Output: a3Kj9mXq2Lnp

## Requirements
- Python 3.x
- No external libraries needed
